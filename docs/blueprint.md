# **App Name**: ANGY Platform Prototype

## Core Features:

- Client Authentication: Secure login/registration for clients with a simple form.
- Client Dashboard: Display of upcoming appointments and order history for clients.
- Service Scheduling: Allows clients to book services via an intuitive calendar interface.
- Service Management: Allows staff to update and review customer orders.
- Inventory Management: Module to monitor and update inventory levels; alerts for low stock.
- Admin Dashboard: A central dashboard with key reports and alerts for admins.
- User Role Management: Module to manage roles of clients, employees and admins. Including options for user approvels.

## Style Guidelines:

- Primary color: Vibrant blue (#0D6EFD) for main navigation and primary actions, conveying trust and action.
- Background color: Light blue (#F0F8FF), nearly desaturated, to provide a clean and spacious feel.
- Accent color: Yellow-Orange (#FFC107) for critical CTAs, drawing immediate attention.
- Text color: Dark Gray (#212529) for high contrast and readability.
- Font pairing: 'Poppins' (sans-serif) for headlines, giving a modern, precise feel, and 'PT Sans' (sans-serif) for body text, balancing modernity with warmth. Note: currently only Google Fonts are supported.
- Use Material Design Icons for clear, universal representation alongside text labels.
- Prioritize a minimalist design with generous whitespace to reduce visual noise.
- Ensure a consistent design language across all components for predictability.
- Implement immediate feedback through subtle animations like toasts or spinners upon user actions.
